Примеры тест-кейсов

"Регистрация на сайте"

Предусловие: тестовые данные. Email не использовался ранее для регистрации на сайте и отсутствует в БД. Пароль должен содержать как минимум 9 символов.

Шаги:

  1. Нажать на ссылку регистрации. ОР: открылась форма регистрации.
  2. В форме регистрации заполнить поле с email. ОР: не возникло ошибки ввода email.
  3. Заполнить поле с паролем. ОР: не возникло ошибки ввода пароля.
  4. Заполнить поле с повторением пароля. ОР: не возникло ошибки ввода повторения пароля.
  5. Нажать на кнопку регистрации.

Ожидаемый результат: пользователь зарегистрирован и занесён в базу данных. Получено сообщение об успешной регистрации. Отправлено письмо на почту пользователя.

"Поиск товара через строку поиска"

Предусловие: нет.

Шаги:

  1. Ввести в строку поиска существующую позицию товара. ОP: появится подсказка при вводе.
  2. Нажать на кнопку поиска.

Ожидаемый результат: вводимый в строку поиска товар отобразился на странице.

"Авторизация пользователя"

Предусловие: пользователь зарегистрирован на сайте.

Шаги:

  1. Нажать на ссылку авторизации. ОР: открылась форма авторизации.
  2. В форме авторизации заполнить поле с email. ОР: не возникло ошибки ввода email.
  3. Заполнить поле с паролем. ОР: не возникло ошибки ввода пароля.
  4. Нажать на кнопку авторизации.

Ожидаемый результат:пользователь аторизован, может видеть историю покупок и личный кабинет.

"Авторизация пользователя, если забыл пароль"

Предусловие: пользователь зарегистрирован на сайте.

Шаги:

  1. Нажать на ссылку авторизации. ОР: открылась форма авторизации.
  2. Нажать на кнопку восстановления пароля. ОР: открылась форма восстановления пароля.
  3. В появившейся форме заполнить поле с email. ОР: не возникло ошибки ввода email.
  4. Нажать на кнопку смены пароля.

Ожидаемый результат:пользователь получает письмо на почту с инструкциями и возможностью задать новый пароль.

"Добавление товара в корзину со страницы с товаром"

Предусловие: зайти на страницу с описанием товара.

Шаги:

  1. Нажать на кнопку добавления товара.

Ожидаемый результат:сообщение об успешном добавлении товара в корзину. Товар появился в корзине.

"Покупка товара из корзины без оплаты"

Предусловие: товар добавлен в корзину. Пользователь авторизован.

Шаги:

  1. Перейти в корзину. ОР: откралась страница с товарами в корзине.
  2. Нажать на кнопку оформления заказа. ОР: открылась форма оформления заказа.
  3. Ввести адрес доставки. Нажать на кнопку далее. ОР: не возникло ошибки ввода адреса. Стал доступен следующий шаг оформления.
  4. Выбрать способ доставки. Нажать на кнопку далее. ОР: не возникло ошибки выбора способа доставки. Стал доступен следующий шаг оформления.
  5. Оставить поля оплаты пустыми. Нажать на кнопку далее.

Ожидаемый результат: нет возможности перейти дальше по процессу покупки.
